# PRODUCT.md — PersonalBrain

## Objective
A personal "second brain" dashboard for a developer who is also a concert photographer. It centralizes creative work tracking, daily coding practice, and on-demand AI assistance in a single, calm workspace.

## Users
- Primary: the owner (technical, visual, values craft and speed).
- Secondary: none — this is a private tool.

## Core modules
1. **Kanban Photo** — track concert shoots through 4 stages: Shooted → Selecting → Editing → Delivered.
2. **LeetCode Companion** — maintain a daily problem-solving streak, get contextual code help.
3. **AI Chat** — a persistent, terminal-style assistant for code reviews, photography tips, and algorithms.

## Tone & feel
- Calm, focused, premium.
- No AI slop: no sparkle emojis, no purple glow overload, no infinite decorative animations.
- Beautiful without shouting.
- A tool you leave open all day.

## Anti-references
- Generic SaaS dashboards.
- ChatGPT clone interfaces.
- Bright neon cyberpunk.
- Material Design cards with heavy shadows.

## Principles
- Information density over whitespace theater.
- Color used sparingly as wayfinding, not decoration.
- Monospace for metadata, labels, and code.
- Everything clickable gets `cursor-pointer`.
- Transitions 200–300ms, never infinite.
